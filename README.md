# :book: Introduction

Welcome to the project repository for **Team 01 - The African Royals**. Here you will find all of the artifacts, presentations, documentation, and source code for our SWE 3313 class project.

We will create a small e-commerce website using Java, SpringBoot, and SQLite to sell one-of-a-kind African jewelry.

# Jewelry Store Final Presentation
- Video 1: https://www.loom.com/share/1bf78ee79ec34572b4e3234ddd1f2ce1
- Video 2: https://www.loom.com/share/26038e77fa9c4f3b8cbd2690e2fbb3ae

- Our implementation met our design on both frontend and backend. When we designed our prototype in Marvel, we had a certain vision that we wanted to come to life, and we made that happen through html and css. We learned that it is very valuable to give youreslf enough time to fix errors without rushing last minute. Upon implementing this project, we came to realize that the hardest part of this project was not each team member's individual code, but integrating everyone's code into a single branch and running the program successfully. This taught us how important it is to add documentation for your code. Although our implementation was a success, we need to practice time management as a team.
